API 명세서
  • File DB에 입력하기
    • 1. Request
    URL : GET /v1/train/insert
    Host : http://localhost:8080
    2. Response
    Name Type Descript
    code String File -> DB 입력의 성공여부 코드 값 (성공:Succes, 실패: Error)
    3. Sample
    HTTP/1.1 200 OK
    Content-Type: application/json:charset=UTF-8
    {
    "code" : "success"
    }
  • 일평균 인원이 가장 많은 상위 10개의 역명과 인원수를 출력하는 API
    • 1. Request
    URL : GET /v1/train/topten
    Host : http://localhost:8080
    2. Response
    Name Type Descript
    train_name String 상위 10개 역명
    cnt String 일 평균 인원 수
    3. Sample
    HTTP/1.1 200 OK
    Content-Type: application/json:charset=UTF-8
    {
    "train_name" : "걍남",
    "cnt" : 202174
    } ...
  • 월 인원수 평균이 가장 낮은 역 명과 인원수를 출력하는 API
    • 1. Request
    URL : GET /v1/train/lowlist
    Host : http://localhost:8080
    2. Response
    Name Type Descript
    train_name String 가장 낮은 역명
    cnt String 월 평균 인원 수
    3. Sample
    HTTP/1.1 200 OK
    Content-Type: application/json:charset=UTF-8
    {
    "train_name" : "둔촌오륜",
    "cnt" : 38809
    } ...
  • 월 인원수 최대 최소의 차이가 가장 큰 역 명을 출력하는 API
    • 1. Request
    URL : GET /v1/train/difflist
    Host : http://localhost:8080
    2. Response
    Name Type Descript
    train_name String 월 인원수 최대 최소 차이가 큰 역 명
    cnt String 월 인원수 최대 최소 차이 수
    3. Sample
    HTTP/1.1 200 OK
    Content-Type: application/json:charset=UTF-8
    {
    "train_name" : "강남",
    "cnt" : 1350567
    } ...
